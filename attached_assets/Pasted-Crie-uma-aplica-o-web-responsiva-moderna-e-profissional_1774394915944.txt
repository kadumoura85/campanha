Crie uma aplicação web responsiva, moderna e profissional para gestão de campanha política municipal.

OBJETIVO
O sistema deve ser um CRM eleitoral hierárquico com:
- dashboards por perfil
- foco territorial (regiões)
- mapa estratégico
- agenda compartilhada
- lembretes internos
- identidade visual personalizada da campanha

O sistema deve ser:
- extremamente simples para uso no celular
- rápido para cadastrar pessoas
- claro para tomada de decisão

STACK
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (auth + banco)
- Recharts (gráficos)

----------------------------------------
PERFIS DO SISTEMA

1. super_admin
2. vereador
3. coordenador_geral
4. coordenador_regional
5. lider

----------------------------------------
HIERARQUIA

vereador / coordenador_geral
→ coordenador_regional
→ lider
→ contatos

----------------------------------------
REGRAS DE ACESSO

- líder vê apenas seus contatos
- coordenador_regional vê apenas sua região e líderes
- coordenador_geral vê toda a campanha
- vereador vê tudo
- apenas líderes e coordenadores_regionais cadastram contatos
- login direciona automaticamente para o dashboard do perfil

----------------------------------------
LOGIN

- login por telefone ou email
- senha simples
- redirecionamento automático para dashboard

----------------------------------------
NÍVEIS DOS CONTATOS

Apenas 3 níveis:
- contato
- simpatizante
- fechado

Interface simples (botões grandes)

----------------------------------------
REGIÕES

Tabela de regiões com:
- nome
- cor
- coordenador_responsavel

- observacao_estrategica

Funções:
- marcar região como prioritária
